## Sobre

Trabalho com desenvolvimento de software focado em backend, automação de processos e construção de sistemas com atenção a escalabilidade, organização de código e experiência do usuário. Tenho interesse particular em arquitetura de sistemas, containerização e produtos SaaS — do desenho técnico à entrega funcional.

Valorizo código legível, decisões de arquitetura bem justificadas e soluções que resolvem problemas reais com o mínimo de complexidade desnecessária.
